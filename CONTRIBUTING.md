# Contributing to Image Caption Generator

Thank you for considering contributing to the Image Caption Generator! We welcome contributions from the community to help improve the project.

## How to Contribute

1. **Fork the repository** on GitHub.
2. **Clone your fork** to your local machine:

   ```bash
   git clone https://github.com/yourusername/image-caption-generator.git
   ```

3. **Create a new branch** for your feature or bug fix:

   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **Make your changes** and commit them with clear and descriptive messages:

   ```bash
   git commit -m "Add feature: description of feature"
   ```

5. **Push your changes** to your fork:

   ```bash
   git push origin feature/your-feature-name
   ```

6. **Submit a pull request** to the main repository.

## Coding Standards

- Follow the existing coding style and conventions.
- Write clear and concise commit messages.
- Ensure that your code is well-documented with comments and docstrings.

## Pull Request Guidelines

- Ensure that your pull request is based on the latest code from the `main` branch.
- Include a detailed description of your changes and the problem they solve.
- Reference any relevant issues in your pull request description.

## Reporting Issues

If you encounter any issues or bugs, please report them by opening an issue on GitHub. Provide as much detail as possible to help us understand and resolve the issue.

Thank you for your contributions!
